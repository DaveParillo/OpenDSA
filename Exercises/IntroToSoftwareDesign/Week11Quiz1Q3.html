For this question, refer to the following lines of code:

  int[][] arr = new int[10][20];
  arr[2][3] = 3;

What element has been set to 3?

The fourth element in the third array.
  • The second element in the third array.
  • The third element in the second array.
  • The third element in the fourth array.

Remember indexing for arrays starts at 0
