The underlying data structure for the NetvidsDatabase class was an array. What is a limitation of using an array concerning capacity?

An array has a fixed length, so if you add more than the the declared amount of movies to the structure, the array needs to be expanded and to do that, a new array needs to be created.
  • You can access each element by it index.
  • When you remove an item you have to shift all the other items over it.
  • The first element is indexed by 0 and not 1.
